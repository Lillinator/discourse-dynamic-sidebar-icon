## Discourse Dynamic Sidebar Icon
This theme component allows admins to change the hamburger bars icon sidebar icon with the state of the sidebar.  The default component setting is the `bars` to open the sidebar and `xmark` to close it, but admins can set those to whatever fontawesome icon they wish.  Icons that are not not in the core Discourse app must be added to the `SVG icon subset` admin setting.  This component has no effect in mobile viewport size.
